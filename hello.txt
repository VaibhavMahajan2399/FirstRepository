Hello GIT

1. git status			        ->	Displays the status of your working directory
2. git add <file name>	        ->	Add a file to the staging area for commit
3. git add .                    ->  Adds all the file to the staging area for commit
4. git commit -m ""		        ->	Save the staged changes into the local repository
5. git log				        ->	Shows the commit history of your respository
6. git diff				        ->	Shows the differences between the changes in files
7. git branch <branch name>     ->  Creates a New branch
8. git branch                   ->  List all branches in respository
9. git checkout <branch name>   ->  Switching working directory to specific branch
10. git merge <branch name>     ->  Join specificed branch name into your current branch that your working on.


BRANCHING AND MERGING

1. git branch <branch name>     =   Created New branch
2. git checkout <branch name>   =   Switched to the New branch from master
3. Create new and modified files
4. git add .                    =   Add all the files in to the stage on New branch
5. git commit -m "message"
6. git checkout master          =   Switched to the master branch from New branch
7. git merge <branch name>      =   Joined New Branch into the master branch
